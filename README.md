# yymmdd

Simple DSL for working with dates.

## Usage

```crystal
require "yymmdd"

include YYMMDD

p yyyy #=> 2016
p yy   #=> 16
```

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  yymmdd:
    github: veelenga/yymmdd
```
